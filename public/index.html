how this works

Telemetry can't prove value — only cost and leverage. So ROI = your assumptions × measured output, shown transparently.

Auto-estimate (default): hours ≈ (edits×min/edit + commands×min/command + asks×min/ask). Tune the knobs above — they're saved in this browser.

Override per session: type the real hours it saved in the sessions table; that replaces the estimate for that row. ● = rated by you.

value = hours × rate. leverage = value ÷ token cost. Nothing here is fabricated — the knobs are visible and yours.

Estimates and per-session ratings are stored in this browser only (localStorage).
